# Schemas

Сюда складывать OpenAPI schema файлы для GPT Actions.

Ожидаемые файлы:

- `railway-action.openapi.yaml` — schema для Railway GraphQL Action.
- `n8n-bridge-action.openapi.yaml` — schema для mcpo/n8n bridge Action.

Не добавлять секреты в schema. Auth должен храниться в настройках GPT Action.
